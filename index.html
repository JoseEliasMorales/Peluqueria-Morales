SERVICIOS

Color Global

Un color que aplica a todo el cabello de manera uniforme.

Balayage

Degrade natural que aporta una gran luminosidad y dimensión. No se distingue dónde empieza y acaba el color.

Mechas Flamboyage

Tecnica Balayge aplicada a mechas con el fin de crear naturalidad, combinando el tono base en las puntas.

Baño de color

Técnica para devolver el color a mechas anteriormente hechas. Recomendable hacerlo despues de un mes de realizadas.

Mechas con Gorra

Ideales para tan solo iluminar un poco el cabello o quedar super rubia.

Mechas Localizadas

Mechas muy definidas. Gruesas, medianas o finas. Muchas o pocas. Creando un estilo para cada personalidad.
